# Selenium Automation Testing Project

## 📌 Project Overview
This project contains automation test scripts developed using Selenium WebDriver and Java for testing the Guru99 Telecom Demo Application.

## 🌐 Application Under Test
https://demo.guru99.com/telecom/index.html

## 🧪 Test Scenarios Covered
- Add Customer
- Add Tariff Plan
- Add Tariff Plan to Customer
- Pay Billing

## 🛠️ Tools & Technologies Used
- Java
- Selenium WebDriver
- ChromeDriver
- Eclipse IDE

## ▶️ How to Run the Project
1. Open project in Eclipse
2. Ensure Selenium libraries are added
3. Run the following classes:
   - AddCustomerTest.java
   - AddTariffTest.java
   - AddTariffToCustomerTest.java
   - PayBillingTest.java

## ✅ Expected Outcome
- Browser launches successfully
- Each flow executes correctly
- Success messages are displayed
